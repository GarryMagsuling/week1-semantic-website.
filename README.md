# RESTOGAR

This is my 4-page website for Restogar, a simple Filipino restaurant. I built it using clean HTML5 and made sure it is easy for everyone to use.

What is on each page:
# index(Home): Welcomes customers and talks about our fresh ingredients.
# about (About Us): Shows our motto and has a clean table showing our open hours.
# menu(Menu): Classic MunggoIngredients: Traditional savory mung bean stew cooked with garlic, onions, and tomatoes, enriched with pork pieces and topped with fresh green leaves. Best paired with steamed rice. 
# Classic Pork Adobo Ingredients: Tender pork chunks braised in soy sauce, vinegar, garlic, bay leaves, and whole peppercorns, served with steamed white rice.
# Sinigang na Manok Ingredients: Tender pieces of chicken simmered in a sour, savory tamarind broth loaded with radish, sitaw (long beans), eggplant, and fresh leafy greens.
# Beef Kare-Kare: Slow-cooked beef portions in a thick, rich peanut sauce, served with local vegetables and savory bagoong (shrimp paste) on the side.
# Sizzling Sisig Tacos Ingredients: Three soft corn tortillas filled with crispy pork sisig, onions, cilantro, and a squeeze of calamansi
# chef.html (Meet the Chef): Chef's Motto "Ang pagluluto ay hindi pabebe, lahat masarap pati ako."

# Accessibility features I used:
- No divs! I only used semantic tags like header, nav, main, section, article, and footer.
- You can navigate the whole site just by using the Tab key on your keyboard.
- The hours table has caption, thead, and th tags so screen readers can easily read it.
- All form inputs are connected directly to their labels using IDs.
- Every image has clear alt text explaining what is in the photo.
